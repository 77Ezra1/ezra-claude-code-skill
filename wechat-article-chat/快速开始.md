# 微信公众号文章助手 - 快速开始

## 最简单的使用方法

### Windows用户（推荐）

1. **双击 `启动中文版.bat`**
   - 会打开一个交互式菜单
   - 选择 `[1]` 保存文章
   - 粘贴文章链接
   - 按回车即可

2. **或双击 `保存文章.bat`**
   - 直接输入文章链接
   - 自动保存

### 命令行用户

```bash
# 中文进度界面
python scripts/save_article_visual_cn.py "文章链接"

# 交互式菜单
python scripts/wechat_cli_cn.py
```

## 需要Cookie？

大部分公开文章**不需要**Cookie。

如果遇到以下情况，才需要配置Cookie：
- 提示"链接已过期"
- 提示"需要登录才能查看"
- 文章仅限关注者可见

配置方法：选择菜单 `[3]` 或查看帮助文档。

## 常见问题

**Q: 中文显示乱码？**
- A: 双击批处理文件启动，或使用 `--theme simple`

**Q: 保存在哪里？**
- A: 默认 `D:/WeChatArticles/`，可在脚本中修改

**Q: 如何批量保存？**
- A: 编写Python脚本调用 `save_article_with_progress()`
